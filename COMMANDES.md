## Scripts Complets
```bash
npm run dev              # Développement
npm run build            # Build production
npm run start            # Serveur production
npm run migrate:check    # Analyse
npm run migrate:auto     # Migration React Router
npm run migrate:client   # Ajouter "use client"
npm run migrate:all      # Migration complète
```

## Résolution Rapide
- **Module non trouvé** → Redémarrer le serveur
- **Erreur hooks** → Ajouter `"use client"`
- **SVG ne marche pas** → Retirer `?react`
